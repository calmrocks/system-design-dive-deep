# Consistent Hashing - Learning Materials

## Articles & Blogs

- [Consistent Hashing: Algorithmic Tradeoffs](https://medium.com/@dgryski/consistent-hashing-algorithmic-tradeoffs-ef6b8e2fcae8) - Deep dive into different consistent hashing algorithms and their trade-offs
- [Consistent Hashing in System Design](https://www.toptal.com/big-data/consistent-hashing) - Comprehensive guide with visual explanations
- [Amazon DynamoDB Paper](https://www.allthingsdistributed.com/files/amazon-dynamo-soho2007.pdf) - Original Dynamo paper explaining consistent hashing in production
- [Jump Consistent Hash](https://arxiv.org/abs/1406.2294) - Google's paper on a fast, minimal memory consistent hash algorithm

## Technical Documentation

- [Cassandra Architecture: Consistent Hashing](https://cassandra.apache.org/doc/latest/cassandra/architecture/dynamo.html) - How Cassandra implements consistent hashing
- [Redis Cluster Specification](https://redis.io/docs/reference/cluster-spec/) - Redis's approach to hash slots and cluster sharding

## Key Concepts to Explore

- Hash ring and key placement
- Virtual nodes (vnodes) for load balancing
- Replication with successor nodes
- Data migration during rebalancing
- Jump consistent hashing vs ring-based hashing
