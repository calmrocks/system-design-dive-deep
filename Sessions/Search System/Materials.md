## Key Concepts to Explore

- Inverted index structure and BM25 scoring
- Text analysis pipeline (tokenizers, filters, analyzers)
- Near-real-time indexing via CDC or outbox pattern
- Scatter-gather query execution across shards
- Autocomplete, fuzzy matching, and faceted search
